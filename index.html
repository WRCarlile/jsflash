Terms

  • JavaScript (JS)
  • JavaScript (JS)

    • JavaScript is the programming language of HTML and the Web.

  • Operators
  • Operators

    • Operators in JavaScript are very similar to operators that appear in other programming languages. The definition of an operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

  • Variables
  • Variables

    • Variables in JavaScript behave the same as variables in most popular programming languages (C, C++, etc) do, but in JavaScript you don't have to declare variables before you use them. If you don't know what declaring is, don't worry about it. It isn't important!

  • Variable naming conventions
  • Variable naming conventions

    • When choosing a variable name, you must first be sure that you do not use any of the JavaScript reserved names. Another good practice is choosing variable names that are descriptive of what the variable holds. If you have a variable that holds the size of a shoe, then name it "shoe_size" to make your JavaScript more readable.

  • Functions
  • Functions

    • A function is a piece of code that sits dormant until it is referenced or called upon to do its "function". In addition to controllable execution, functions are also a great time saver for doing repetitive tasks.

  • Methods
  • Methods

    • A method is an action that a number can perform or have taken on it.

  • Arguments
  • Arguments

    • Some methods take one or more arguments that provide the method with additional information to help it perform its action.

  • Parameters
  • Parameters

    • Function parameters are the names listed in the function definition.

  • Return
  • Return

    • The return value is the method's response.

  • Chaining methods
  • Chaining methods

    • Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

  • Strings
  • Strings

    • A JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

  • Booleans
  • Booleans

    • A binary variable, having two possible values called 'true' and 'false'.

  • Undefined
  • Undefined

    • When a variable doesn't have a value.

  • NaN
  • NaN

    • Not a number.

  • Escape
  • Escape

    • The escape() function encodes a string.

  • Alert
  • Alert

    • An alert box is often used if you want to make sure information comes through to the user.

  • Comments
  • Comments

    • JavaScript comments can be used to explain JavaScript code, and to make it more readable.

  • JQuery
  • JQuery

    • A cross-platform JavaScript library designed to simplify the client-side scripting of HTML. jQuery is the most popular JavaScript library in use today, with installation on 65% of the top 10 million highest-trafficked sites on the Web. jQuery is free, open-source software licensed under the MIT License.
